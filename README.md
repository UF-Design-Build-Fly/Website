# UF Design/Build/Fly Website

Modernized React site for Gator Aeronautics / UF Design/Build/Fly.

## Local development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

The site uses the existing GitHub Pages homepage path:
`https://UF-Design-Build-Fly.github.io/Website`
